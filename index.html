Research

My current research focus on object detection and network compression, specifying in small/tiny/occluded object detection as well as model compression, including quantization, pruning and distillation. My research uses techniques from classical optimization, feature engineering and sometimes takes inspiration from cognitive science.

Confidence-driven Bounding Box Localization for Small Object Detection
Huixin Sun, Baochang Zhang, Yanjing Li, Xianbin Cao
Under Review, 2023
code / arXiv

We present a novel Confidence-driven Bounding Box Localization (C-BBL) method to rectify the gradient distortion phenomenon in small object detection.
